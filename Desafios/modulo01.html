Tag <Picture>

Usar esta tag quando tiver diferentes tamanhos de imagens, para a imagem se adaptar em diferentes dispositivos.

Dentro da tag <pictures> colocara a imagem maior na tag <img>, e acima desta colocar a tag <source:media:type>, onde a imagem menor sempre deve vir primeiro no código.

CSS e seus estilos.

CSS InLine

Fica na mesma linha que as tags HTML, deixando assim o código muito poluido se for colocado muita informação, como cor, tipo de fonte, e etc. Além de ser muito mais demorado, entretanto é um bom método para usar em ocasiões específicas como colocar determinado parágrafo a direita.

CSS Interno

Este cria uma tag style dentro de <head>, e a partir disso você pode ir criando seu código CSS muito mais organizado, separado do conteúdo. Porém caso seja necessário colocar o mesmo estilo em diversas páginas, pode acabar se tornando um processo massante, ainda mais se necessário fazer alguma alteração.

CSS Externo

Este cria uma nova página no código chamada de style.css, podendo ser renomeada, consiste em você fazer todo o código CSS em uma página separada e após pronto, usar a tag link:css, este é muito mais prático com relação aos outros e também o mais usado, por se tratar de um arquivo excluisivo para estilo.
